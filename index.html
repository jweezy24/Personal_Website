Current Projects

Icon

VoltKey

Voltkey is a project started by one of my advisors Dr. Klingensmith. He had the idea of creating a physical password for a family routing system. My role in this project is developing the cryptographic mechanism for their communication. We centered around a Reed-Solomon based algorithm which, with our research, is able to be tunable for any house.

Icon

Privacy Work

Recently I have done quite a few projects studying privacy. I was published at PoPETS 2022 for my work with user audio privacy. The paper explored a user's understanding of the mute button and how we should implement privacy for native applications that run on the operating system. In that work, we discovered that Webex maybe leaking more information about a user then they intended too. If you click the icon above, it will take you to a repository for that work.

Icon

Moonshine

Moonshine is a project for context based authentication systems to create better randomness from their low entropy data sources. A lot of sensor based entropy sources have weak entropy. When using Moonshine, good random numbers can still be extracted from the source.
