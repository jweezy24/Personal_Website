VoltKey

VoltKey is a distributed systems project as well as a cryptographic project. My role in Voltkey was to examine the randomness in the bit streams and design more effective measures to manage the randomness. This is a C project.

FLIC

FLIC is a distributed systems project. The project focuses on utilizing fog computing and attempts to add an API to abstract fog systems to increase bandwidth efficency. My role was the lead researcher. FLIC is a Python 3 project.

Non-Local Mean Curvature

The topic of Non-Local Mean Curvature is an anaylitical mathematical question which mathmaticians have been more recently interested in. There is a need for a simulation of this mathematical question. Therefore, I have developed a simulation tool which can calculate the curvature of a curve. The research lies in developing the estimation for unbounded curves.
